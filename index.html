  • How to create an s3 bucket and evable versioning in AWS.

Why use bloated WordPress and pay monthly fees to host a portfolio website when AWS gives you 1 million free lambda requests a month then charges $0.20 per 1m requests thereafter? This sounds great and all but the learning curve for hosting a website solely in Lambda is pretty steep. Serving images, js, CSS, favicon, and all your assets from a Lambda function and Api Gateway isn’t all that straightforward. I’m here to show you how we can do this with little effort and in just two Lambda functions. I’ll be using Serverless Framework, Python3.6, a template I purchased from themefrost.net, and a domain name purchased through Route53. We’re using AWS, which means you need an AWS account. The Serverless Framework link provided above has a guide and video on how to set up your account.
